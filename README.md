# 🚌 Simulador de Terminal Rodoviário

Este repositório contém um **simulador de terminal rodoviário** desenvolvido em **linguagem C**.  
O projeto foi realizado como **trabalho final da disciplina de Programação e Algoritmos**, no **1º período de Análise e Desenvolvimento de Sistemas (ADS)**.  

---

## 📖 Estrutura do projeto
O código foi desenvolvido em **um único arquivo `.c`**, contendo diversas **funções** responsáveis por operações específicas do sistema.  
A função `main()` atua apenas como **ponto de entrada**, chamando as demais funções, o que garante uma separação lógica mesmo sem múltiplos arquivos.  

A persistência de dados é feita em **arquivos binários**, permitindo salvar e recuperar informações como motoristas, viagens e reservas de passagens.  

---

## 🚀 Funcionalidades
- Cadastro de motoristas  
- Cadastro e controle de frotas  
- Reserva de passagens (com verificação de poltronas ocupadas)  
- Área de login para motoristas  
- Persistência de dados em arquivos binários
